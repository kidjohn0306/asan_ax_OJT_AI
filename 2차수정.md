# 2차 수정 사항 — Admin UI 구조 개편

## 현재 문제점 분석

### 현재 사이드바 구조
```
대시보드
시험 관리
  ├─ 시험 생성 (exam-create)   ← AI 문제 생성 + 시험지 저장이 한 화면에 혼재
  └─ 검토·수정 (exam-review)  ← 검토 대기 문제 승인/반려
응시 이력
문제 관리                      ← 실질적으로 문제은행인데 이름만 "문제 관리"
사용자 승인
결과 분석
설정
```

### 문제점
- `시험 생성` 화면(ExamCreate)에서:
  - 좌측: 팀/난이도/문항수 설정 → `AI 문제 생성` 버튼
  - 우측: AI 생성 결과 미리보기 → `시험지 저장` 버튼
  - **AI로 문제를 새로 만드는 행위**와 **검증된 문제로 시험지를 구성하는 행위**가 하나의 화면에 섞여 있음
- `문제 관리`는 문제은행 역할(목록 조회, 난이도 조정, 승인/반려)을 하고 있으나 명칭이 불명확

## 목표 구조

### 개념 분리

| 메뉴 | 역할 | 데이터 흐름 |
|---|---|---|
| **문제 생성** | Claude AI API 호출 → 문제 초안 생성 | AI → draft |
| **검토·검증** | 생성된 초안을 관리자가 승인/반려 | draft → reviewing → approved/rejected |
| **문제은행** | 승인된 문제 목록 확인, 난이도 조정 | approved 문제 열람 |
| **시험지 생성** | 문제은행에서 검증 완료 문제로 시험지 구성 | approved → 시험지 |

### 변경 후 사이드바 구조 (안)
```
대시보드

문제 관리 그룹
  ├─ 문제 생성    ← AI API 호출, 생성 옵션(팀/카테고리/수량)
  ├─ 검토·검증   ← 생성된 초안 승인/반려 (현재 exam-review와 동일)
  └─ 문제은행    ← 승인된 문제 목록 + 난이도 조정

시험 관리 그룹
  └─ 시험지 생성  ← 문제은행 문제 선택 → 시험지 구성 → 저장

응시 이력
사용자 승인
결과 분석
설정
```

## 화면별 변경 내용

### 1. 문제 생성 (신규 분리)
현재 `exam-create`의 좌측 절반에서 분리.
- 팀 선택, 카테고리, 난이도, 문항 수 설정
- `AI 문제 생성` 버튼 → 생성 후 "검토·검증" 탭 안내 토스트
- 생성 직후 결과 미리보기는 인라인으로 표시 (시험지 저장 버튼 없음)

### 2. 검토·검증 (exam-review → 이름만 변경, 로직 유지)
현재 `ExamReview` 컴포넌트 구조 그대로.
- 검토 대기 문제 목록 (좌) + 문항 상세/승인/반려 (우)

### 3. 문제은행 (questions → 이름·역할 명확화)
현재 `Questions` 컴포넌트 기반.
- 상태 필터 탭: 전체 / 승인 / 검토대기 / 반려 / 초안
- 난이도 조정 드롭다운 유지
- **시험지 생성으로 이동** 버튼 추가 (문제은행과 시험지 생성 연결)

### 4. 시험지 생성 (신규 분리)
현재 `exam-create`의 우측 절반 + 시험지 구성 로직.
- 문제은행에서 승인된 문제 목록을 팀/카테고리/난이도 필터로 조회
- 문제 선택 → 시험지 구성 미리보기
- `시험지 저장` / `PDF 생성` 버튼

## 확인된 사항

1. **검토·검증 탭 위치** → `문제 관리 그룹` 하위로 이동 ✅
2. **시험지 생성 UI** → 팀/난이도 조건으로 **자동 배분** 후, 해당 화면에서 문제 순서 변경·개별 문제 교체 등 **피드백 편집** 가능 ✅
3. **문제은행 기본 필터** → 진입 시 `승인(approved)` 문제만 기본 표시 ✅
4. **대시보드 빠른 실행 버튼** → 메뉴 구조 변경에 맞춰 같이 수정 ✅

## 시험지 생성 상세 플로우

```
① 조건 설정 (팀 / 난이도 비율 / 문항 수)
        ↓
② 자동 배분 → 시험지 미리보기 (문제 목록 + 순서)
        ↓
③ 피드백 편집
   - 문제 순서 변경 (드래그 or ▲▼ 버튼)
   - 개별 문제 교체 (문제은행에서 동일 난이도 문제로 스왑)
   - 문제 삭제
        ↓
④ 시험지 저장 / PDF 생성
```
