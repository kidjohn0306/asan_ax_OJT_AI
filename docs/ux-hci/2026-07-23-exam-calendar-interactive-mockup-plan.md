# 시험 관리·달력 인터랙티브 목업 계획

## 2026-07-23

### 목적

시험 관리·달력 고도화 분석안이 실제 관리자 화면에 적용됐을 때의 구조와 상호작용을 독립 실행형 HTML 목업으로 검증한다.
이번 단계에서는 React, FastAPI, 저장소 계약을 수정하지 않는다.

### 구현 범위

- 운영 우선순위와 향후 일정을 함께 보여주는 대시보드.
- 월 달력과 목록 보기를 전환하는 시험 생성·관리 화면.
- 팀, 유형, 버전, 문항 구성으로 비교하는 시험지 선택 팝업.
- 오늘, 이번 주, 날짜 선택을 지원하는 응시 현황 화면.
- 시험 생성 패널의 시험지 선택, 일정 설정, 응시자 확인 흐름.
- 데스크톱과 태블릿 너비의 반응형 배치.

### 상호작용

- 좌측 메뉴로 목업 화면을 전환한다.
- 달력 날짜를 선택하면 해당 날짜의 시험 목록이 바뀐다.
- 빈 날짜에서 시험 생성 패널을 열 수 있다.
- 시험지 선택 팝업에서 팀 필터, 목록 선택, 상세 미리보기를 수행한다.
- 선택한 시험지의 팀이 시험 생성 패널에 읽기 전용으로 반영된다.
- 시험 관리에서 달력 보기와 목록 보기를 전환한다.
- 응시 현황에서 날짜 범위, 팀, 상태 필터를 변경한다.

### 시각 원칙

- 현재 관리자 화면의 흰색 카드, 짙은 남색 사이드바, 파란 강조색을 유지한다.
- 정보 계층과 운영 우선순위를 개선하되 장식 효과는 절제한다.
- 색상만으로 상태를 구분하지 않고 텍스트와 배지를 함께 제공한다.
- 날짜와 주요 컨트롤은 키보드 포커스가 보이도록 구성한다.

### 검증 기준

- 대시보드, 시험 관리, 응시 현황으로 이동할 수 있다.
- 시험 관리에서 날짜를 선택하고 시험 생성 패널을 열 수 있다.
- 시험지 선택 팝업에서 팀별 시험지를 비교하고 선택할 수 있다.
- 선택한 시험지의 팀이 임의 변경 불가 상태로 표시된다.
- 달력과 목록 전환이 정상 동작한다.
- 응시 현황의 빠른 날짜 필터와 상세 카드가 동작한다.
- 390px와 1024px 너비에서 주요 콘텐츠가 가로로 넘치지 않는다.
